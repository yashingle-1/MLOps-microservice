# AWS-Ready MLOps Microservice 

## Overview
A production-ready **MLOps project** that deploys machine learning models via a **FastAPI microservice**, containerized with **Docker**, managed with **Poetry**, and deployed automatically to **AWS EC2** using **GitHub Actions CI/CD**.

---

## Tech Stack
- **Backend & ML Serving:** Python, FastAPI  
- **Containerization & Orchestration:** Docker, Docker Compose  
- **CI/CD:** GitHub Actions  
- **Cloud Deployment:** AWS EC2  
- **Dependency Management:** Poetry  
- **Testing:** Pytest for unit testing  
- **Model Serialization:** Pickle  

---

## Features
- Serve ML models in production via REST API  
- Containerized service with Docker for scalability  
- Automated CI/CD for testing and deployment  
- Reliable unit-tested API endpoints
